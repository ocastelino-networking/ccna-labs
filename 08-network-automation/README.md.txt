# Network Automation Lab (Python Basics)

## Objective
Introduce automation in networking using Python scripts.

---

## Concept
Instead of manually configuring routers and switches, Python can automate repetitive tasks.

---

## Use Case
- Push configs to multiple devices
- Collect device information
- Reduce human error

---

## Skills Practiced
- Python basics for networking
- Simulated configuration automation
- Understanding CLI automation workflow

---

## Outcome
Basic understanding of how automation is used in modern networks.
