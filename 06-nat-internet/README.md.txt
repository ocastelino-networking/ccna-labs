# NAT Configuration Lab

## Objective
Enable private network devices to access the internet using NAT.

---

## Scenario
- Inside network: 192.168.1.0/24
- Outside (Internet): 200.1.1.0/24

Devices must access external networks using a single public IP.

---

## Key Concept
Network Address Translation (NAT):
- Translates private IPs to public IPs
- Conserves IPv4 addresses

---

## Skills Practiced
- NAT overload (PAT)
- Inside/outside interface configuration
- IP translation

---

## Outcome
Private devices can access external networks via one public IP.
