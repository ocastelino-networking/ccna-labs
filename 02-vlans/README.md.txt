# VLAN Configuration Lab

## Objective
Learn how to segment a network using VLANs on a Cisco switch.

## Scenario
A company has 3 departments:
- Sales (VLAN 10)
- HR (VLAN 20)
- IT (VLAN 30)

Each department must be isolated from the others for security and traffic control.

---

## Key Concepts
- VLAN (Virtual Local Area Network)
- Access Ports
- Switch Port Assignment
- Broadcast Domain Segmentation

---

## Skills Practiced
- Creating VLANs on a switch
- Assigning switch ports to VLANs
- Understanding Layer 2 segmentation

---

## Outcome
Devices in different VLANs cannot communicate unless routing is configured.
