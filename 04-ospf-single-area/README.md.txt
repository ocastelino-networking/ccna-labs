# OSPF Single Area Lab

## Objective
Configure dynamic routing between multiple routers using OSPF.

---

## Scenario
Three routers are connected:

- Router A → 192.168.1.0/24
- Router B → 192.168.2.0/24
- Router C → 192.168.3.0/24

All routers must learn each other's networks automatically.

---

## Protocol Used
OSPF (Open Shortest Path First)

---

## Key Concepts
- Dynamic routing
- Link-state protocol
- Area 0 (backbone area)
- Route advertisement

---

## Outcome
Routers automatically share routing information without static routes.
