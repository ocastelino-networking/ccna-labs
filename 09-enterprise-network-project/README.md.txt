# Enterprise Network Infrastructure Project

## Objective
Design and simulate a small enterprise network using core CCNA technologies.

---

## Network Overview

Departments:
- VLAN 10 = Sales
- VLAN 20 = HR
- VLAN 30 = IT

Features Implemented:
- VLAN segmentation
- Inter-VLAN Routing
- OSPF dynamic routing
- ACL security filtering
- NAT/PAT internet access
- DHCP automatic IP assignment
- Basic Python automation concepts

---

## Technologies Used
- Cisco IOS
- OSPF
- VLANs
- ACLs
- NAT/PAT
- DHCP
- Python
- Git & GitHub

---

## Skills Demonstrated
- Enterprise network design
- Routing and switching
- Security implementation
- Network automation fundamentals
- Technical documentation
- Infrastructure troubleshooting

---

## Outcome
Successfully designed a scalable enterprise-style network capable of secure interdepartmental communication and simulated internet connectivity.
