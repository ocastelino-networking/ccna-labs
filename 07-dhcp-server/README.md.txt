# DHCP Configuration Lab

## Objective
Configure a router to automatically assign IP addresses to clients using DHCP.

---

## Scenario
Network: 192.168.10.0/24

Clients should automatically receive:
- IP address
- Subnet mask
- Default gateway

---

## Key Concept
DHCP eliminates manual IP configuration by automating address assignment.

---

## Skills Practiced
- DHCP pool configuration
- Excluding IP addresses
- Gateway assignment

---

## Outcome
Clients receive IP addresses automatically when connecting to the network.
